CSS控件底部横线绘制方法

多种实现方式及效果演示

方法一：使用 border-bottom

最简单直接的方法，通过设置底部边框实现横线效果。

效果演示

CSS代码

.input-border {
    width: 100%;
    padding: 10px;
    border: none;
    border-bottom: 2px solid #3498db;
    background: transparent;
    font-size: 16px;
    transition: border-color 0.3s;
}

.input-border:focus {
    outline: none;
    border-bottom-color: #e74c3c;
}

方法二：使用伪元素 (::after)

通过伪元素创建横线，提供更大的灵活性。

效果演示

CSS代码

.input-pseudo {
    width: 100%;
    padding: 10px;
    border: none;
    background: transparent;
    font-size: 16px;
    position: relative;
}

.input-pseudo::after {
    content: '';
    position: absolute;
    left: 0;
    bottom: 0;
    width: 100%;
    height: 2px;
    background: #3498db;
    transition: background 0.3s;
}

.input-pseudo:focus::after {
    background: #e74c3c;
}

方法三：使用 box-shadow

利用阴影效果创建底部横线，可以实现更复杂的效果。

效果演示

CSS代码

.input-shadow {
    width: 100%;
    padding: 10px;
    border: none;
    background: transparent;
    font-size: 16px;
    box-shadow: 0 2px 0 #3498db;
    transition: box-shadow 0.3s;
}

.input-shadow:focus {
    outline: none;
    box-shadow: 0 2px 0 #e74c3c;
}
